# SANDBOX WordPress e2e testing

## prerequisite

- Node.js
- direnv (optional)
- Yarn (optional)

## usage

```shell
$ git clone git@github.com:kamataryo/sandbox-wordpress-e2e-testing.git
$ cd sandbox-wordpress-e2e-testing
$ yarn # or `npm install`
$ cp .envrc.sample .envrc
$ vi .envrc
$ npm test
```
